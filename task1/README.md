# Задание №1 (Правила Iptables)
## Постановка задачи
Необходимо разработать и применить конфигурацию правил для утилит `iptables` и `ip6tables`. Сетевой экран должен полностью блокировать входящий трафик, но разрешать исходящий. Дополнительно требуется обеспечить доступность хоста для сетевой диагностики с помощью утилиты `ping`.

---

## Результат
Любой входящий трафик блокируются на уровне ядра. Локальные сетевые процессы функционируют в штатном режиме, беспрепятственно получая ответы на исходящие запросы. Хост корректно обрабатывает `ping`, а текущие и новые сессии удаленного администрирования по SSH защищены от разрыва и блокировки.

---

## Принцип реализации

* **Контроль состояний:** Первым активным правилом разрешается входящий трафик со статусом `ESTABLISHED,RELATED`. Это позволяет ядру динамически сопоставлять входящие пакеты с уже открытыми исходящими сессиями, что обеспечивает возврат трафика (например, ответов от веб-серверов при запросах через `curl`) и гарантирует стабильность текущей SSH-сессии.
* **Локальный интерфейс и доступ:** До ввода ограничений принудительно разрешается трафик на интерфейсе локальной петли (`lo`), необходимый для корректного межпроцессного взаимодействия внутри ОС. Также открывается входящий TCP-порт 22 для обеспечения возможности будущих подключений по SSH.
* **Специфика IPv6 (Neighbor Discovery):** В конфигурации `ip6tables` отдельно разрешаются низкоуровневые пакеты ICMPv6 (`router-advertisement`, `neighbor-solicitation`, `neighbor-advertisement`). В сетях IPv6 они выполняют функции протокола ARP. Их явное одобрение критично, так как без них хост теряет связь со шлюзом провайдера на сетевом уровне.
* **Диагностика ICMP:** В цепочку `INPUT` добавлено точечное разрешение на пакеты `echo-request`, что позволяет хосту принимать запросы `ping` и автоматически генерировать ответы.
* **Политика по умолчанию (`DROP`):** В финальной части конфигурации для цепочек `INPUT` и `FORWARD` устанавливается политика по умолчанию `DROP`. Любой сетевой пакет, не соответствующий ни одному из вышеперечисленных разрешающих критериев, уничтожается на входе.
